Pi agent (pi.dev) by Mario Zechner. Open-source, unopinionated, TypeScript-based agent coding tool. Extension stacking model: each extension registers hooks, tools, commands independently. Composes via CLI flags. No built-in subagent support, teams, or safety modes. The inverse of Claude Code's opinionated defaults. Worth monitoring as a hedge tool and as a source of architectural patterns for halos extension design. Source: https://youtu.be/f8cfH5XX-XU?si=tpHpFnmy3ucwyU43
